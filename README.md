修正:
- マイルート選択直後は「走行前」状態
- ルート全体、スタート位置、ゴール/チェックポイントを確認してからスタート

- 距離と自転車アイコンは1秒タイマーでRPMに応じて自動進行
- 最高速度は現在速度の最大値として更新
- 一時停止中は距離/時間/消費が進まない
- 100%到達で自動的に完走リザルト表示
- 通常の日本一周モードでも手入力中は自動進行

D66-61 Model Explain + Ranking Fairness

D66-60ベース。
実車感覚モデルの違いをUIで説明し、ランキング公平性の設計方針を明示。

追加:
- 設定画面に「モデルごとの違い」カード
  - 各モデルの60rpm / 80rpm / 100rpm想定速度
  - 漕ぎやめ後の減速タイプ
  - 短い説明
- 設定画面に「ランキングの公平性」カード
  - 旅の距離 = 選択モデルで進むリアル優先
  - ランキング距離 = 全員同じ標準モデルで換算する公平優先
- calcOfficialRankingKmhFromRpm(rpm)
  - ランキング用速度は選択モデルに依存しない
  - ひとまずcross相当の標準モデルを固定基準にする

ランキング設計方針:
- ロードバイク選択で旅の体感距離が伸びるのは現実再現として許可
- ただし公式ランキングではロード選択だけでは有利にしない
- 将来のランキングでは「旅km」と「公式km」を分ける
- 必要ならモデル別ランキングも併設可能

安全策:
- 走行ロジックは変更しない
- センサー解析は変更しない
- 地図描画は変更しない
- 保存/読込/編集系関数は触らない
